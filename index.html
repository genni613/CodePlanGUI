快速开始

三步上手

无需注册，无需绑定账号，几分钟即可开始使用。

1

安装插件

从 GitHub 下载 zip，Settings → Plugins → Install from Disk

2

配置 Provider

Settings → Tools → CodePlanGUI，填入你的 API Endpoint 和 Key

3

开始对话

打开右侧 Tool Window，开始和 AI 畅聊你的代码
