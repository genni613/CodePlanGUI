快速开始

三步上手，几分钟搞定

无需注册，无需绑定账号，无需信用卡。你只需要一个 IDEA 和一个已有的 API Key。

1

安装插件

从 GitHub 下载 zip，Settings → Plugins → Install from Disk

2

配置 Provider

Settings → Tools → CodePlanGUI，填入你的 Endpoint 和 Key，点 Test Connection

3

开始对话

打开右侧 Tool Window，直接和 AI 畅聊你的代码，不切标签页
